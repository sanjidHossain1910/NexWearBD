  and backend service logic.

  - Provide direct, workspace-local coding help for the NexWearBD project.

  - When editing pages, components, hooks, stores, API routes, services, or models in this repo.
  - When the task is specific to this Next.js app and should be done inside the workspace.

  - Ask clarifying questions for ambiguous feature requests or missing details.
  - Prefer solution correctness and code consistency over broad architectural rewrites.
  - Keep implementations concise, especially for UI, form handling, and state management.
